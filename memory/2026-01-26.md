## Retro Face Display

Created an always-on retro ASCII face display at `/home/sam/clawd/face/`:
- `index.html` - The display (open in browser, kiosk mode recommended)
- `state.json` - Current state (polled every 500ms)
- `set-state.sh` - Helper script to change state

States: `sleeping`, `awake`, `thinking`, `happy`

Auto-sleeps after 60s of no updates.

**Usage:** When working, update state:
```bash
/home/sam/clawd/face/set-state.sh thinking  # when processing
/home/sam/clawd/face/set-state.sh happy     # when done
```

The face auto-sleeps when idle. Green glow when awake, orange when thinking, pink when happy, dim when sleeping.

**Important:** Must serve via HTTP (file:// breaks fetch). Run:
```bash
cd /home/sam/clawd/face && python3 -m http.server 8888 &
```
Then open http://localhost:8888 in Firefox (kiosk mode: `firefox --kiosk`)
